# Example Controlled Blueprint: GitHub Repo Intelligence Agent

This example shows how Matrix Builder should hand a controlled task to AI coders.

Files:

- `MATRIX_BLUEPRINT.yaml`: the locked architecture.
- `MATRIX_STANDARDS.lock.yaml`: the applied standards and RMD rules.
- `validation-report.json`: expected validation output.
- `standards-report.md`: human-readable compliance report.
- `prompts/`: AI-coder prompt examples.

Use this example as the first integration test for `agent-generator`.
