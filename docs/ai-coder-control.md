# AI-Coder Control

Matrix Builder mitigates uncontrolled AI coding by generating:

- `MATRIX_BLUEPRINT.yaml`
- `MATRIX_STANDARDS.lock`
- `MATRIX_TASKS.md`
- `MATRIX_ALLOWED_CHANGES.md`
- `MATRIX_ACCEPTANCE_CRITERIA.md`
- `MATRIX_VALIDATION.md`

AI coders receive task-scoped prompts with allowed files, forbidden changes, validation commands, and acceptance criteria.
