# Exception approval process

Exceptions are allowed only through pull request review.

## Required review

- Critical rule exception: Ruslan Magana approval required.
- Security rule exception: security maintainer approval required.
- MatrixHub publishing exception: MatrixHub maintainer approval required.

## Required fields

- rule id
- scope
- reason
- compensating control
- owner
- expiration date

Exceptions without an expiration date are rejected.
